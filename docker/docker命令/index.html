docker 命令

June 19, 2021

一、docker 基础命令#

# 更新容器自动重启
docker update --restart=always 容器名称

# 进入容器
docker exec -it 容器名称 /bin/bash

# 获取容器ip
docker inspect --format='{{.NetworkSettings.IPAddress}}' 容器名称

二、docker 启动容器#

# 启动InfluxDB
docker run --name influxdb -p 8086:8086 influxdb:2.0.7
