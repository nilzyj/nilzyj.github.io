Hexo 使用记录

[Hexo使用记录]

1、添加本地图片

第二步：将图片放入markdown目录下，使用方式。（hexo new “markdown.md”会在_post下生成markdown目录和markdown.md）

2、markdown开头描述中要使用英文分号

3、配置记录：gitssl认证暂时关闭，git config –system http.sslverify falsed
